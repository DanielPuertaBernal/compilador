# ── Bonus Entrega 4: Integración de IA con Groq ──────────────────────────────
# El bonus NO requiere librerías externas de Python (usa urllib estándar).
#
# Solo necesitas una clave de API gratuita de Groq (sin tarjeta de crédito):
#   1. Ir a: https://console.groq.com
#   2. Iniciar sesión y ve a "API Keys" → "Create API Key"
#   3. Copia la clave (empieza con "gsk_...")
#   4. Crea un archivo ".env" en la raíz del proyecto (copia .env.example)
#      y pega tu clave: GROQ_API_KEY=gsk_...
#
# Configurar la clave manualmente (alternativa sin archivo .env):
#   Windows PowerShell:  $env:GROQ_API_KEY = "gsk_..."
#   Linux / macOS:       export GROQ_API_KEY="gsk_..."
#
